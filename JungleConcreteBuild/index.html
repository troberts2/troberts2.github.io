About This Project

Jungle Concrete was a prototype with the given prompt: Climate Change. We tried to create a game where you build plant buildings to decrease pollution and factories to earn money and increase pollution. The goal is to find a balance to win the game.

For this project I mostly worked on the tile placing systems as well as the building placing and upgrading systems.
